===========================================
HERO SLIDER IMAGES - SETUP INSTRUCTIONS
===========================================

To add images to the Hero Slider:

1. Save your images in this folder (assets/images/hero-slider/)

2. Name your images as:
   - image1.jpg (or .jpeg, .png, .webp)
   - image2.jpg
   - image3.jpg
   - ... and so on up to image10.jpg

3. Supported image formats:
   - .jpg or .jpeg
   - .png
   - .webp

4. Image naming rules:
   - Must start with "image" (lowercase)
   - Followed by a number from 1 to 10
   - Example: image1.jpg, image2.png, image3.webp

5. How it works:
   - The slider will automatically detect which images are available
   - Only images that exist in this folder will be displayed
   - If you have only 4 images (image1.jpg, image2.jpg, image3.jpg, image4.jpg),
     only those 4 will be shown in the slider
   - Missing images (e.g., image5.jpg through image10.jpg) will be automatically skipped

6. Recommended image specifications:
   - Resolution: 1920x1080 pixels or higher (for best quality)
   - Aspect ratio: 16:9 (landscape)
   - File size: Optimize images to reduce loading time (under 500KB each recommended)

7. To update images:
   - Simply replace the existing image files with new ones
   - Keep the same file names (image1.jpg, image2.jpg, etc.)
   - Clear your browser cache to see the updated images

8. To remove an image:
   - Delete the image file from this folder
   - The slider will automatically hide that slide

===========================================
EXAMPLE:
===========================================

If you have these files in this folder:
- image1.jpg ✓
- image2.png ✓
- image3.jpg ✓
- image4.jpg ✓

Then the slider will show only 4 slides (image1, image2, image3, image4)
and automatically create 4 navigation dots.
